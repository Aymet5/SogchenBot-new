# Система управления молебнами (Telegram Бот + Админ-панель)

Это руководство поможет вам развернуть проект на вашем собственном сервере (VPS/VDS) под управлением Ubuntu/Debian.

## Требования к серверу
- ОС: Ubuntu 20.04 / 22.04 или Debian
- Установленный Node.js (версия 18 или 20)
- Установленный Git

## Шаг 1: Подготовка сервера

Подключитесь к вашему серверу по SSH и выполните следующие команды для установки необходимых программ (если они еще не установлены):

```bash
# Обновляем пакеты
sudo apt update && sudo apt upgrade -y

# Устанавливаем Node.js (версия 20) и npm
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt install -y nodejs

# Устанавливаем PM2 для поддержания работы бота 24/7
sudo npm install -g pm2

# Устанавливаем инструменты для сборки (нужны для базы данных SQLite)
sudo apt install -y build-essential python3
```

## Шаг 2: Загрузка кода на сервер

Создайте папку для проекта и скачайте код из вашего GitHub:

```bash
# Переходим в домашнюю директорию
cd ~

# Клонируем ваш репозиторий (ЗАМЕНИТЕ ССЫЛКУ НА СВОЮ)
git clone https://github.com/ВАШ_ЛОГИН/ВАШ_РЕПОЗИТОРИЙ.git datsan-bot

# Переходим в папку с проектом
cd datsan-bot
```

## Шаг 3: Установка зависимостей и настройка

Находясь в папке `datsan-bot`, выполните установку библиотек:

```bash
npm install
```

Создайте файл конфигурации `.env`:

```bash
nano .env
```

Вставьте в него ваш токен от Telegram-бота:
```env
TELEGRAM_BOT_TOKEN="8513945043:AAGxwiiUOHFbGfsa0DLYHrMt-SODcF53RuM"
```
*(Чтобы сохранить файл в nano: нажмите `Ctrl+O`, затем `Enter`, затем `Ctrl+X` для выхода).*

> **Примечание:** Переменную `APP_URL` на сервере указывать **не нужно**. Без неё бот автоматически перейдет в режим `polling` (опрос серверов Telegram), что идеально подходит для серверов без настроенного домена и SSL-сертификата.

## Шаг 4: Сборка фронтенда (Админ-панели)

Соберите панель управления:

```bash
npm run build
```
Эта команда создаст папку `dist` с готовыми файлами сайта.

## Шаг 5: Запуск проекта

Теперь запустим сервер с помощью PM2, чтобы он работал в фоновом режиме и автоматически перезапускался при сбоях или перезагрузке сервера:

```bash
# Запускаем сервер
pm2 start npm --name "datsan-bot" -- start

# Сохраняем список процессов PM2, чтобы они запускались после перезагрузки сервера
pm2 save
pm2 startup
```
*(Команда `pm2 startup` выдаст вам строку, которую нужно скопировать и выполнить в терминале).*

## Шаг 6: Проверка работы

1. **Бот:** Откройте Telegram и напишите вашему боту `/start`. Он должен ответить.
2. **Админ-панель:** Откройте в браузере IP-адрес вашего сервера с портом 3000:
   `http://IP_ВАШЕГО_СЕРВЕРА:3000`

## Как обновлять код в будущем

Если вы внесете изменения в код на GitHub и захотите обновить их на сервере, выполните эти команды в папке проекта (`cd ~/datsan-bot`):

```bash
# Получаем новые изменения из GitHub
git pull

# Устанавливаем новые зависимости (если появились)
npm install

# Пересобираем админку (если меняли дизайн)
npm run build

# Перезапускаем сервер
pm2 restart datsan-bot
```

## Полезные команды PM2

- `pm2 logs datsan-bot` — посмотреть логи (ошибки и сообщения сервера).
- `pm2 status` — проверить статус работы.
- `pm2 stop datsan-bot` — остановить бота.
